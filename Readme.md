# 💬 QuoteSphere – Quotes Listing & Custom Quote Generator

QuoteSphere is a modern and responsive Quotes Listing Application built using HTML, Tailwind CSS, and Vanilla JavaScript.

The application fetches inspirational quotes dynamically from the FreeAPI Quotes endpoint and also allows users to generate custom motivational quotes using their own input words.

---

# 🚀 Features

- 🔥 Dynamic quote fetching using FreeAPI
- 🎲 Random quote generator
- ✨ Custom quote generation from user input
- 🎨 Professional dark mode UI
- 🔍 Search quotes, authors, and categories
- 🧠 Smart quote categorization
- 📋 Copy quote functionality
- ⚡ Loading state handling
- 📱 Fully responsive design
- 🎯 Category filters:
  - Motivation
  - Success
  - Life
  - Wisdom
- 🔄 Refresh quotes functionality
- 🛡️ Fallback quotes support

# ⚙️ Functionalities

## 📌 Quotes Listing

- Fetches quotes dynamically from FreeAPI
- Displays quotes in professional cards
- Shows author names and categories

---

## 🎲 Random Quote Generator

- Generates random quotes from fetched API data
- Displays random author details
- Refreshes dynamically on button click

---

## ✨ Custom Quote Generator

Users can enter words such as:

```bash
dream success life
```

The application generates motivational quotes dynamically like:

```bash
"Great things begin with dream success life and determination."
```

This feature is implemented using:
- Dynamic user input handling
- Randomized quote templates
- JavaScript template literals

---

## 🔍 Search Functionality

The search system filters:
- Quote text
- Author names
- Categories

---

## 🎯 Category Filters

Quotes are categorized into:
- Motivation
- Success
- Life
- Wisdom

Since the API does not provide categories directly, intelligent frontend categorization has been implemented.
